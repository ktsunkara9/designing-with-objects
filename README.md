# Designing with Objects

A Java Maven project for practicing object-oriented programming and object creation patterns.

## Purpose

This project is intended to practice object creation, design patterns, and object-oriented principles in Java.

## Build and Run

```bash
mvn compile
mvn exec:java -Dexec.mainClass="inc.skt.designingwithobjects.Main"
```

## Test

```bash
mvn test
```
